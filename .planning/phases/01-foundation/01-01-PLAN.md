<task type="auto">
  <name>Task 2: Install python-dotenv and psycopg2-binary</name>
  <files>requirements.txt</files>
  <action>
Install packages in virtual environment:
1. Activate .venv if not active
2. pip install python-dotenv psycopg2-binary
3. Run pip freeze > requirements.txt to capture ALL dependencies with pinned versions

This creates requirements.txt with:
- Django==5.2.10
- python-dotenv (for .env file loading)
- psycopg2-binary (PostgreSQL adapter - binary for easier install)
- All transitive dependencies pinned
  </action>
  <verify>pip list | grep -E "python-dotenv|psycopg2" && cat requirements.txt</verify>
  <done>python-dotenv and psycopg2-binary installed, requirements.txt exists with pinned versions</done>
</task>
